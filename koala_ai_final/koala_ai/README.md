# 🐨 Koala AI

A calm, wise AI assistant with real-time web search — powered by Google Gemini + DuckDuckGo.

## ✨ Features
- 🌐 Real-time web search (DuckDuckGo — free, no API key!)
- 💬 ChatGPT-style chat history sidebar
- 💾 Chats auto-saved in browser
- 🐨 Beautiful Koala / Eucalyptus theme
- 📱 Mobile friendly

## 🚀 Setup

1. Install dependencies:
   pip install -r requirements.txt

2. Create .env file:
   cp .env.example .env
   Then add your Gemini API key inside .env

3. Run:
   python app.py

4. Open: http://localhost:5000
