# 🐼 Panda AI

A calm, wise AI assistant with real-time web search — powered by OpenRouter + DuckDuckGo.

## ✨ Features
- 🌐 Real-time web search (DuckDuckGo — free, unlimited!)
- 💬 ChatGPT-style chat history sidebar
- 💾 Chats auto-saved in browser
- 🐼 Beautiful Panda / Bamboo theme
- 📱 Mobile friendly

## 🚀 Setup
1. pip install -r requirements.txt
2. cp .env.example .env → add your OpenRouter API key
3. python app.py
4. Open: http://localhost:5000
